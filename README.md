# lol-stats

====
Download the League of Legend's match-history and parse to csv.

## Description
You can analyze the LoL Game from match-history.
LJLのスタッツをまとめたいと思いましたが、LJLの試合はRiot APIで情報を取得することができないので、Match-historyから情報を取得するために作りました。
